# AI-Powered Automated HR Manager

Comprehensive AI-powered HR Manager system for mid-sized organizations.

## Project Structure

- `backend/`: FastAPI primary API server.
- `frontend/`: React web application.
- `mobile/`: React Native mobile application.
- `microservices/`: Node.js microservices (Email/Notifications).
- `docs/`: Project documentation.

## Tech Stack

- **Backend:** FastAPI, Python 3.10+, PostgreSQL, MongoDB, Redis, ChromaDB.
- **AI:** LangChain, LangGraph, Gemini API (Generative AI), OpenCV, DeepFace.
- **Frontend:** React 18, TypeScript, Tailwind CSS, Redux Toolkit.
- **Mobile:** React Native.

## Getting Started

1. Clone the repository.
2. Ensure Docker and Docker Compose are installed.
3. Run `docker compose up --build`.

## Modules

1. Employee Management
2. Attendance & Face Recognition
3. Leave Management (AI-driven)
4. Performance Management
5. AI Chat Agent
6. Recruitment
7. Reporting & Analytics
8. Notification System
9. Salary & Compensation Management
10. Automated Payslip Generation
